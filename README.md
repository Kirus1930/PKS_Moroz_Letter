### Запуск проекта
1) Открыть проект в Android Studio (проект использует базу данных SQLite, поэтому лучше запускать на Android-эмуляторе)
2) Перейти в папку проекта
```bash
cd flutter_moroz_letter_eremin
```
3) Запустить проект
```bash
flutter run
```
